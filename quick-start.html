Quick Start

Choose the path that fits your workflow. Each guide takes you from zero to published content on testnet.

New to Morse? Start with Core Concepts for an overview, or jump straight into the CLI guide - it's the fastest way to see Morse in action.

CLI
Install morse-cli from npm, create a publication, publish and read content - all from your terminal. For operators and content managers.
~5 min · Requires: Bun or Node, funded Sui wallet
TypeScript SDK
Wire up @arcadiasystems/morse-sdk, create a publication, add an entry, and read it back in a single script. For app developers building on Morse.
~5 min · Requires: Bun or Node, funded Sui wallet
Smart Contracts
Interact with Morse's deployed contracts directly via PTBs. For developers who want full control over transaction construction.
~10 min · Requires: Sui CLI, funded Sui wallet
