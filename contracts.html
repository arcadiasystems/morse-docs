Contract Reference

Complete reference for the Morse Move package modules, public functions, events, error codes, and deployment information.

Package

Move.toml 
name = "publication"
edition = "2024"

[dependencies]
Sui = { git = "https://github.com/MystenLabs/sui", ... }
Walrus = { git = "https://github.com/MystenLabs/walrus", ... }

The package name is publication. All three modules live within it:

ModulePathResponsibility
publication::publication sources/publication.move Top-level API: publications, capabilities, collection and entry operations, Seal integration
publication::collection sources/collection.move Collection struct, storage modes, entry table management
publication::entry sources/entry.move Entry and revision structs, BlobRef enum, validation logic
Deployment addresses. Testnet deployment addresses are stored in morse-contracts/Published.toml (committed). Ephemeral localnet addresses go to Pub.local.toml (gitignored). Check the repository for the latest published addresses.

Shared Objects

Two objects are shared at the protocol level they must be passed as arguments to relevant transactions:

ObjectCreated byRequired for
PublicationRegistry init() at publish time new_publication, delete_publication
Publication new_publication + share_publication All collection and entry operations, cap management, Seal

Public Functions

publication::publication

FunctionRequired capReturnsDescription
new_publication(registry, ctx, name, slug) (Publication, OwnerCap, PublisherCap) Create a new publication. Register slug in registry.
share_publication(publication) Share the publication object so others can interact with it.
delete_publication(registry, publication, owner_cap) OwnerCap Delete publication and release slug. Collections must be empty.
transfer_owner_cap(owner_cap, recipient) Transfer ownership to another address.
issue_publisher_cap(publication, owner_cap, holder, ctx) OwnerCap PublisherCap Mint a new publisher cap bound to holder.
revoke_publisher_cap(publication, owner_cap, cap_id) OwnerCap Add cap ID to the revocation denylist.
destroy_publisher_cap(publication, cap, ctx) PublisherCap (holder) Voluntarily surrender write access. Cleans up denylist entry if revoked.
create_collection(publication, cap, name, storage_mode, ctx) PublisherCap Add a new named collection to the publication.
delete_collection(publication, cap, name, ctx) PublisherCap Remove and destroy a collection. Entries must be empty first.
add_entry_to_collection(publication, cap, collection_name, name, blob, quilt_patch_id, content_type, encrypted, access_policy, seal_id, ctx) PublisherCap u64 (entry_id) Create a new entry in a named collection.
delete_entry_from_collection(publication, cap, collection_name, entry_id, ctx) PublisherCap Remove an entry by ID.
append_collection_entry_draft_revision(publication, cap, collection_name, entry_id, blob, quilt_patch_id, content_type, encrypted, access_policy, seal_id, ctx) PublisherCap u64 (revision_id) Append a draft revision and advance draft_head.
publish_collection_entry_from_draft(publication, cap, collection_name, entry_id, draft_revision_id, blob, quilt_patch_id, content_type, ctx) PublisherCap u64 (revision_id) Publish from a draft appends public revision, advances public_head.
publish_collection_entry_direct(publication, cap, collection_name, entry_id, blob, quilt_patch_id, content_type, ctx) PublisherCap u64 (revision_id) Publish directly without a draft step.
contains_slug(registry, slug) bool Check if a slug is currently registered.
get_publication_id_by_slug(registry, slug) &ID Resolve a slug to its publication's object ID.

Events

All events have copy, drop abilities and are emitted from publication::publication.

EventFieldsTrigger
PublicationCreated publication: ID, name: String, slug: String new_publication
PublicationDeleted publication: ID, name: String delete_publication
SlugRegistered slug: String, publication: ID Slug claimed in registry (always emitted with PublicationCreated)
SlugReleased slug: String, publication: ID Slug freed from registry (always emitted with PublicationDeleted)
PublisherCapIssued publication: ID, cap: ID issue_publisher_cap
PublisherCapRevoked publication: ID, cap: ID revoke_publisher_cap
CollectionAdded publication: ID, name: String create_collection
CollectionRemoved publication: ID, name: String delete_collection
